ℹ️ ¿Qué es EAN-13?

EAN-13 (European Article Number) es el estándar internacional de código de barras más usado en productos de consumo.

📏13 dígitos: 12 de datos + 1 verificador
✅Requerido por MercadoLibre para mejorar visibilidad
🔢El dígito verificador se calcula automáticamente
⚠️Este generador crea códigos técnicamente válidos para uso interno
📋 Cómo usarlo en ML
1 Generá un código EAN-13 con el botón de arriba
2 Copiá el código con el botón "Copiar"
3 Abrí la publicación en ML → Editar → campo "Código universal (GTIN)" → pegá el código
✓ Los datos no se envían automáticamente a ML — solo es una herramienta de generación
